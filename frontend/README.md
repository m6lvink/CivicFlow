# CivicFlow - Frontend

Next.js 16 web interface for the CivicFlow permit delay predictor and requirements checker.

## Tech stack

- **Language:** TypeScript (`.ts` / `.tsx`) with strict type checking
- **Framework:** Next.js 16.2.7 App Router on React 19.2.4
- **Styling:** Tailwind CSS v4 through `@tailwindcss/postcss`, with custom theme tokens in `app/globals.css`
- **Fonts:** `next/font/google` loading Figtree and Lora
- **Icons:** `lucide-react`
- **API access:** browser calls same-origin `/api/*`; Next route handlers proxy to FastAPI with server-only env
- **Tooling:** npm scripts, TypeScript 5, ESLint 9, `eslint-config-next`

## Prerequisites

- Node 20.9+, npm 10+
- CivicFlow FastAPI backend running at `API_BASE_URL` (defaults to `http://127.0.0.1:8000`)

## Development

```bash
npm install
npm run dev    # http://localhost:3000
```

The dev server hot-reloads on save. Copy `.env.example` to `.env.local` if FastAPI is not on `http://127.0.0.1:8000` or backend auth is enabled.

## Build

```bash
npm run build  # production build
npm run lint   # ESLint
```

## Structure

```
app/
  layout.tsx      # root layout, fonts (Figtree + Lora via next/font)
  login/page.tsx  # shared-password gate for private demos
  page.tsx        # main page: form on left, results on right
  globals.css     # Tailwind v4 design tokens (warm editorial palette)
proxy.ts          # protects / and /api/* when login env is configured
components/
  DropZone.tsx     # PDF/image upload and drag-drop input
  PermitForm.tsx  # manual permit details form
  PredictResult.tsx  # delay forecast panel
  CheckReport.tsx    # DPP requirements checklist
lib/
  api.ts          # typed fetch wrappers for /extract, /predict, /check, /schema
  types.ts        # TypeScript types matching FastAPI response shapes
```

## Render

Deploy this app as a Render Web Service, not a Static Site. `render.yaml` wires the private
backend address and generated API key. Set the remaining server-only values:

```bash
CIVICFLOW_SITE_PASSWORD=<shared demo password>
CIVICFLOW_SESSION_SECRET=<generated by Render>
```

Do not use `NEXT_PUBLIC_*` for API URLs or keys; browser requests should show `/api/...`.
Set both login values before exposing the site. If either login value is set without the
other, the gate fails closed.

## Design

Custom Tailwind v4 tokens defined in `globals.css`. Fonts loaded via `next/font/google`:
- Figtree (sans) - body text and UI labels
- Lora (serif) - numeric display and headings

No shadcn/ui or default Tailwind component library. The design system is custom; don't pull in defaults.
